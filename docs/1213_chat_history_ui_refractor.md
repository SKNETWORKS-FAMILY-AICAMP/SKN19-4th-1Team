# 채팅 이력 관리 UI 구조 개선

**날짜**: 2025-12-13
**작업**: 대화 내역 UI 구조 정리 및 닫기 버튼 동작 개선

## 변경 사항
### 1. history 관련 코드 분리

- chat.js에 있던 대화 기록(history) 관련 HTML, CSS 코드를 각각 chat.html, chat.css로 이동
- JavaScript 파일은 로직 처리만 담당하도록 역할 분리

### 2. 비로그인 사용자 접근 제한

- 비로그인 상태에서 과거 대화 내역 조회 버튼 클릭 시
    -  대화 목록 대신 “로그인 후 이용해주세요” 안내 메시지 출력하도록 수정

### 3. 우측 패널 상태 복원 기능

- 과거 대화 목록의 “닫기” 버튼 클릭 시
    - 우측 패널이 과거 대화 목록 → 기본 결과 메시지 화면으로 전환되도록 수정
    - 초기 디폴트 추천 메시지가 다시 표시됨
