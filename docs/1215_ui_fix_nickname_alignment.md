# 닉네임 변경 입력창 UI 수정

## 개요
사용자 설정 페이지(`settings.html`)에서 닉네임 변경 섹션의 UI 레이아웃을 수정하였습니다.

## 수정 내용
- **대상 파일**: `unigo/templates/unigo_app/setting.html`
- **변경 사항**:
  - 닉네임 입력 필드(`input`)와 중복 확인 버튼(`button`)을 한 줄에 나란히 배치.
  - 이를 위해 두 요소를 `div` 컨테이너로 감싸고 `display: flex` 및 `gap: 10px` 스타일 적용.
  - 버튼의 불필요한 `margin-top` 스타일 제거.

## 해결 과정
- 기존에는 줄바꿈 되어 있던 입력창과 버튼을 직관적으로 연결되어 보이게 하기 위해 Flexbox 레이아웃을 사용.
- 모바일 및 데스크탑 환경에서 일관된 간격을 유지하도록 `gap` 속성 활용.
