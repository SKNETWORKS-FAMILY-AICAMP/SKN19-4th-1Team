# 전공 추천 온보딩 트리거 구현 (2025-12-12)

## 개요
기존에는 채팅방 입장 시 전공 추천을 위한 온보딩 질문(선호 과목 등)이 자동으로 시작되었습니다.
이는 일반적인 대학/입시 정보를 묻고자 하는 사용자에게 혼란을 줄 수 있어, **사용자가 원할 때만 명령어를 통해 추천 과정을 시작**하도록 변경했습니다.

## 변경 사항

### 1. 자동 시작 비활성화
- `init()` 함수에서 온보딩 상태(`onboardingState.isComplete`)가 `false`여도 자동으로 질문을 던지지 않도록 수정했습니다.
- 대신 초기 환영 메시지를 통해 챗봇의 기능을 소개하고, 추천 기능을 사용하는 방법을 안내합니다.

### 2. 트리거 명령어 도입
- **명령어**: "추천 시작"
- **동작**: 채팅창에 "추천 시작"이라고 입력하면, 즉시 선호 교과목을 묻는 1번 질문부터 온보딩 프로세스가 시작됩니다.
- 기존 대화 내역이 있더라도 온보딩 상태를 초기화(`isComplete = false`)하고 새로 시작합니다.

### 3. Frontend 수정 (`unigo/static/js/chat.js`)
- **`init()`**: 자동 시작 로직 제거 및 환영 메시지("전공 추천을 받고 싶다면 '추천 시작'이라고 입력해주세요.") 추가.
- **`handleSubmit()`**: 사용자 입력이 "추천 시작"과 일치할 경우 `startOnboardingStep()`을 호출하는 인터셉트 로직 추가.

## 사용 방법
1. 채팅방에 입장하면 기본적으로 일반 챗봇 모드입니다. ("궁금한 점을 물어보세요!")
2. 전공 추천을 원하면 **"추천 시작"**을 입력합니다.
3. "안녕하세요! 고등학교 과목 중 가장 자신 있거나..." 질문이 나오면 순서대로 답변을 진행합니다.
